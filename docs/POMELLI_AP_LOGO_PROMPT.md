# Pomelli Logo Prompt - Abhijit Pramanik Portfolio

Use this prompt in Google Pomelli or any logo generator.

## Main Prompt

Create a premium personal brand logo for "Abhijit Pramanik", using an "AP" monogram as the main mark. The brand is a proof-backed digital portfolio for SEO, content strategy, WordPress publishing, social media operations, creative assets, and AI/no-code workflow systems. The logo should feel modern, credible, technical, sharp, and recruiter-friendly, while matching the current website direction: bold purple, acid-lime highlights, white space, and strong contrast.

Design psychology: trust, proof, signal, organized intelligence, practical execution, and future-ready digital craft. The mark should communicate that the person can connect search data, content, systems, and real project proof into clear business outcomes. It should feel energetic and memorable, but still clean enough for resumes, a portfolio header, and a browser extension icon.

Visual direction: geometric AP monogram, clean negative space, sharp but approachable edges, subtle signal/path motif, and a slightly futuristic editorial-tech feel. Use a compact symbol that still reads well in a favicon. Optional detail: a slim spark, pulse line, or angled cut that hints at growth, ranking lift, or verified proof without turning the logo into a generic marketing icon.

Color palette:
- Core brand purple: #1A0263
- Acid lime highlight: #CEFF1A
- Crisp white: #FFFFFF
- Deep near-black outline: #111111
- Optional soft lavender accent: #DCCEFF

Style: premium editorial-tech, minimal, high contrast, flat vector, scalable, balanced spacing, no clutter. It should look strong on the current purple website background, but also work on white resume pages and extension UI.

Output requirements:
- Primary AP monogram
- Horizontal lockup with "Abhijit Pramanik"
- Icon-only version
- Dark background and light background versions
- SVG/vector style
- Transparent background option

Avoid:
- Mascots
- Generic AI swirl
- Random lightning icons
- Overused gradient blobs
- Too many colors
- Complex 3D text
- Agency-style "AP Digital" naming
- Anything that looks like a finance, gaming, or crypto logo
- Anything too feminine, too playful, or too corporate-generic

## Short Version

Premium AP monogram for Abhijit Pramanik, a proof-backed SEO, content, WordPress, social media, and AI/no-code workflow portfolio. Modern editorial-tech style, geometric negative space, trust/proof/signal psychology, bold purple #1A0263 with acid-lime #CEFF1A, white, and near-black accents. Create icon, horizontal lockup, dark/light versions, SVG-ready, favicon-safe. Avoid mascots, generic AI swirl, gradient blobs, and agency-looking branding.
